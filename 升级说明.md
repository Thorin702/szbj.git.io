# 平台升级说明 v2.0

## 🎉 重大更新

本次升级大幅提升了平台的数据量和交互体验，主要改进包括：

### 1. 数据大幅扩充

#### 企业数据
- **原版**：5家详细企业
- **升级版**：60家详细企业 + 完整统计数据
- **新增字段**：企业地址、法人代表、网站、浏览量等

#### 政策数据
- **原版**：6条政策
- **升级版**：50条详细政策
- **新增字段**：点击量、申请数、成功率、政策热度等

#### 招商项目
- **原版**：3个项目 + 3个线索
- **升级版**：50个项目 + 50个招商线索
- **新增字段**：项目描述、联系次数、风险分析等

#### 服务案例
- **原版**：3个案例
- **升级版**：20个服务案例
- **新增字段**：满意度、服务周期等

### 2. 文件变更

#### 新增文件
```
frontend/js/data-full.js          # 完整数据文件（替代data.js）
frontend/js/data-extended.js      # 数据生成工具
frontend/index.html.backup        # 原版本备份
```

#### 已更新文件
```
frontend/index.html               # 已更新引用新数据文件
```

### 3. 使用方法

#### 方式一：直接使用（推荐）
当前的index.html已经自动更新引用，直接打开即可看到扩充后的数据。

#### 方式二：切换回旧版本
如果需要切换回旧版本：
```bash
cd frontend
copy index.html.backup index.html
# 将data-full.js改回data.js
```


```javascript
{
    id: 政策编号,
```

### 性能优化

1. **数据懒加载**
   - 页面切换时才加载对应数据
   - 图表按需初始化
   - 大数据集分页展示

2. **缓存机制**
   - 已加载的页面缓存在内存
   - 图表实例复用
   - 减少重复计算

3. **响应式优化**
   - 图表自适应窗口大小
   - 移动端优化布局
   - 懒加载图片和资源

## 📞 技术支持

### 常见问题

**Q：图表不显示怎么办？**
A：
1. 检查浏览器控制台是否有错误
2. 确认ECharts CDN可以访问
3. 尝试刷新页面（Ctrl+F5）

**Q：数据加载很慢？**
A：
1. 检查网络连接
2. 清除浏览器缓存
3. 关闭其他标签页释放内存

**Q：如何添加自定义数据？**
A：
1. 编辑 js/data-full.js
2. 修改对应的数据数组
3. 刷新页面查看效果
