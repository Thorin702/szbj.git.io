## 八、常用控制台命令速查

```javascript

enterpriseData.list.length

// 政策数量

// 项目数量

```

### 数据筛选
```javascript
// 查找海淀区企业
enterpriseData.list.filter(e => e.district === '海淀区')

// 查找人工智能企业
enterpriseData.list.filter(e => e.type === '人工智能')

// 查找高成长企业
enterpriseData.list.filter(e => e.growth === 'high')

// 查找财政补贴政策
policyData.list.filter(p => p.type === '财政补贴')
```

### 数据分析
```javascript
// 计算平均营收
enterpriseData.list.reduce((sum, e) => sum + e.revenue, 0) / 60

// 统计各区企业数
const count = {};
enterpriseData.list.forEach(e => {
    count[e.district] = (count[e.district] || 0) + 1;
});
console.table(count);
```

## 十、遇到问题？

### 问题1：控制台在哪里？
**答**：按 **F12**（或右键点击页面 → 检查），切换到 **Console** 标签

### 问题2：输入命令后没反应？
**答**：
1. 确认已按回车
2. 检查是否有错误提示（红色文字）
3. 刷新页面重试（Ctrl+F5）

### 问题3：数据量没有增加？
**答**：
1. 确认打开的是最新的index.html
2. 检查浏览器控制台，输入 `enterpriseData.list.length`
3. 应该显示60，如果不是，清除缓存重试

### 问题4：政策解读窗口不显示？
**答**：
1. 检查是否有弹窗拦截
2. 检查控制台是否有错误
3. 刷新页面重试
