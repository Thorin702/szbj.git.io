# 北京市数字产业发展支撑平台 - 前端演示系统

## 项目简介

本项目是"数据创新反哺数字产业发展"综合性数字产业发展支撑平台的前端演示系统，基于纯HTML+CSS+JavaScript实现，无需任何npm配置或后端服务，可直接在浏览器中打开使用。

## 功能特点

### 1. 总览页面
- **产业统计看板**：展示北京市28条重点产业链、15,847家规上企业等核心数据
- **产业分布图表**：可视化展示各产业产值分布（ECharts饼图）
- **增长趋势分析**：展示2019-2023年企业数量和产值规模变化
- **创新能力评估**：各产业企业数量对比分析
- **产业链卡片**：6大重点产业链快速浏览和详情查看

### 2. 招前分析模块
- **产业图谱分析**
  - 6大产业（集成电路、人工智能、新能源汽车、生物医药、5G通信、工业互联网）
  - 每个产业包含：企业数量、产值规模、增长率、完整度评分
  - 点击可查看详细产业链结构（上游-中游-下游）
  - 产业链图谱可视化（ECharts关系图）

- **企业发展分析**
  - 5家示例企业详细数据
  - 企业画像：注册资本、营收、纳税、员工、专利等
  - 企业评分：创新能力、成长性、信用等级
  - 企业雷达图可视化分析

- **强链补链分析**
  - 强链机会：人工智能、集成电路等优势产业建议
  - 补链机会：新能源汽车动力电池、生物医药高端器械等薄弱环节
  - 延链机会：元宇宙、商业航天等新兴方向

### 3. 招中匹配模块
- **政策匹配引擎**
  - 选择企业自动匹配适用政策
  - 智能评分系统（匹配度百分比）
  - 6大类政策：财政补贴、税收优惠、人才引进、融资支持等
  - 政策详情：条件、优惠、申报流程

- **政策设计工具**
  - 项目基本信息录入
  - 预期效益评估
  - 自动生成政策组合方案

- **招商项目管理**
  - 3个示例项目（谈判中、已签约、已落地）
  - 项目进度可视化
  - 投资额、预期产值、预期税收、就业人数

### 4. 招后服务模块
- **金融服务**
  - 4类金融产品：科技创新贷、知识产权质押贷、上市培育贷、供应链金融
  - 产品详情：额度、利率、期限、特点、要求
  - 服务案例时间轴展示

- **人才服务**
  - 人才招聘服务（本地人才库25,000+人）
  - 人才培训服务（156家机构、2,340+门课程）
  - 人才政策匹配

- **产业对接**
  - 供需智能匹配工具
  - 产业对接活动列表
  - 活动状态跟踪

- **监测预警**
  - 企业健康度统计（健康/预警/风险）
  - 风险预警案例
  - 预警类型和处理状态

### 5. 数据中台
- **数据资产目录**
  - 企业数据（工商、税务、社保、信用）
  - 产业数据（产业链、市场、技术、人才）
  - 政策数据（国家、地方、行业政策）

- **数据应用**
  - 大模型服务
  - 知识图谱
  - BI分析

## 使用方法

### 方法1：直接打开（推荐）
1. 找到 `frontend/index.html` 文件
2. 双击文件，使用默认浏览器打开
3. 开始体验平台功能

### 方法2：通过文件路径打开
1. 打开浏览器
2. 按 `Ctrl+O` (Windows) 或 `Cmd+O` (Mac)
3. 选择 `frontend/index.html` 文件
4. 点击打开

### 方法3：本地服务器（可选）
如果需要更好的开发体验，可以使用简单的HTTP服务器：

**Python 3:**
```bash
cd frontend
python -m http.server 8000
# 访问 http://localhost:8000
```

**Node.js (需先安装http-server):**
```bash
npx http-server frontend -p 8000
# 访问 http://localhost:8000
```

## 常见问题

### Q: 打开页面后图表不显示？
A: 检查网络连接，确保可以访问ECharts CDN (cdn.jsdelivr.net)。如无法访问，可下载ECharts到本地并修改引用路径。

### Q: 点击按钮没有反应？
A: 打开浏览器控制台(F12)查看是否有JavaScript错误。确保三个JS文件都正确加载。

### Q: 图表显示不全或错位？
A: 调整浏览器窗口大小触发响应式布局。图表会自动适应窗口大小。

### Q: 如何修改主题颜色？
A: 在 `<style>` 标签中搜索颜色值（如 `#667eea`），替换为自定义颜色。

### Q: 可以连接真实后端吗？
A: 可以。修改 `js/data.js`，将数据改为从API获取。例如：
```javascript
async function loadIndustryData() {
    const response = await fetch('/api/industry');
    const data = await response.json();
    return data;
}
```
